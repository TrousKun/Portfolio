Statement of purpose

- แรงจูงใจที่ทำให้ผมเลือกคณะ "วิทยาการคอมพิวเตอร์"

เนื่องจากว่าผมต้องการเรียนการ Coding จากที่ผมได้ลองและผมรู้สึกชอบวิธีการทำงานของโปรแกรม รู้สึกสนุกไปกับการ Coding

 

- ผมจึงอยากที่จะรู้ถึงการ Coding ขั้นสูงที่การศึกษาด้วยตัวเองตามแหล่งที่มาเช่น YT หรือ ตาม Website ต่างๆ ไม่สามารถมอบให้ผมได้

และสาเหตุสำคัญของการที่ผมเลือกที่จะฝึกการเขียน Code คือ การที่ตลาดของประเทศและโลก อาชีพอย่าง Programmer เป็นที่ต้องการในตลาดค่อนข้างสูง

ผมจึงอยากที่จะเรียนรู้สิ่งนี้เพื่อนำไปใช้พัฒนาองค์กรที่ผมอาจจะสร้างขึ้นในอนาคต หรือ ไปช่วยองค์กรอื่นพัฒนาการเพื่อตามทันโครงสร้างของโลกที่เปลี่ยนไปอย่างรวดเร็ว ณ ปัจจุบัน

เนื่องจากผมเรียนรู้ด้วยตัวเองมาส่วนนึงผ่าน YT หรือ website อย่าง w3school

ผมสามารถเขียน Basic of HTML หรือ Command พื้นฐานของ Python

และกำลังศึกษาเพิ่มเติมเกี่ยวกับ CSS ที่ผมใช้ ณ ตอนนี้ที่พอทำเป็นมีแค่การทำเป็นแถบเหมือนที่เห็นนี้

 

- และมีอีก 1 สิ่งที่ผมเคยทำที่พอเป็นประสบการณ์สำหรับการเขียน เว็ปของผมได้เรียกว่าเป็นประสบการณ์ครั้งแรกกับการเขียนเว็ปจริงๆจังๆของผมเลยก็ว่าได้

นั่นคือการที่ทางบริษัทที่แม่ของผมกำลังทำงานอยู่ได้มีการเปลี่ยนชื่อบริษัท

และจำเป็นจะต้องจ้าง Programmer มาเขียนเว็ปให้ใหม่เพื่อแก้ข้อมูลให้เป็นปัจจุบัน

ผมจึงอาสาที่จะขอลองที่จะเขียนเองโดยใช้ Model เว็ปเดิมมาเป็นแบบเนื่องจากเขาต้องการเว็ปแบบเดิมแต่เปลี่ยนข้อมูลภายในเว็ป

และผมก็ทำมันสำเร็จถือเป็นประสบการณ์การทำงานของครั้งแรกกับการ Coding ของผมโดยใช้พื้นฐานต่างๆที่ศึกษามาด้วยตัวเองนำมาผสมผสานกัน

 

และสุดท้ายนี้สิ่งที่ผมอยากจะบอก

หากผมสามารถเข้าไปศึกษาการทำงานขั้นสูงภายในมหาลัยได้ผมจะสามารถทำงานได้อย่างมีประสิทธิภาพมากขึ้นและสามารถพัฒนาองค์กรให้มีประสิทธิภาพมากยิ่งขึ่นไปอีกได้
